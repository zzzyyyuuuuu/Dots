before you start

Ensure you are on an Arch-based distro with Hyprland support. The script handles dependencies.

1
Clone the repo to your home directory
2
Run the install script — links configs
3
Log out and select Hyprland
4
Done. Hit SUPER + D

quick install

bash — vroomies install
# clone the repo
git clone https://github.com/zzzyyyuuuuu/vroomies ~/.config/vroomies
 
# navigate and run
cd ~/.config/vroomies
chmod +x install.sh
./install.sh
